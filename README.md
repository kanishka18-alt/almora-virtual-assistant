# Almora – Virtual Assistant
Almora is a **voice-enabled personal assistant** that allows users to interact using both **text and speech**.
It can answer queries, perform quick commands, and fetch relevant information from the web using integrated APIs.

## Features
* Voice and text-based interaction
* Real-time speech recognition and speech synthesis
* Web search results using Google Custom Search API
* Quick commands for common tasks (time, date, opening websites)
* Secure API key management using environment variables
* Error handling for reliable responses

## Tech Stack
* Node.js
* Express.js
* JavaScript
* Google Custom Search API
* Web Speech API (SpeechRecognition & SpeechSynthesis)
* dotenv

## Project Structure
public/- Static assets 
src/- Frontend application 
server/- Backend API and server code

## Environment Variables
Create a `.env` file in the root directory and add the following:
GOOGLE_SEARCH_API_KEY=your_api_key_here
GOOGLE_SEARCH_ENGINE_ID=your_search_engine_id

These variables are used to securely access the **Google Custom Search API**.
